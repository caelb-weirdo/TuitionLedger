# TuitionLedger

Simple tuition management for tutors: students, classes, browser-approved QR attendance, fees, and WhatsApp reminders.

## Apps

- `tutor-frontend` — Vite + vanilla JavaScript tutor dashboard
- `student-mobile` — small installable student PWA
- `backend` — lightweight Flask API
- `supabase/schema.sql` — database schema

Run each app from its directory. Copy `.env.example` to `.env` and fill in local values.
